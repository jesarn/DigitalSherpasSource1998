MICROSOFT IE USERS HELP PAGE

This site uses Macromedia's Flash2 Plugin. It will automatically download to you computer via an ActiveX script. If you received a security message while downloading the Plugin you will need to set your security to MEDIUM. To do this follow the following steps:

Step 1: Click on View in the menu bar

Step 2: Next, click on Options...

Step 3: Click on the "Security" tab

Step 4: Click on the Safety Level... button

Step 5: Finally, select the Medium setting

After that all you need to do is "Reload" the page and it will download the plug-in automatically! Once you have it installed you can visit other sites that use Flash2 and you won't have to wait for the plug-in download.
